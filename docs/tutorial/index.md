# Tutorial
{: .no_toc }

A step-by-step walkthrough from understanding the crossover design to adapting the toolkit for your own teaching context.
{: .fs-6 .fw-300 }

---

This tutorial guides you through the entire process of using the crossover-edtech-toolkit to plan, execute, and analyze a crossover experiment in your classroom. Each step builds on the previous one, so we recommend following them in order.

**Expected time:** 30--45 minutes (longer if you run the code alongside the tutorial).

## The 6 steps

| Step | Title | What you will learn |
|:-----|:------|:--------------------|
| 1 | [Understanding the Crossover Design](01-study-design) | What a crossover (AB/BA) design is, why it is powerful, and the statistical model behind it |
| 2 | [Setting Up Your Environment](02-setup) | How to install R or Python dependencies, clone the repo, and verify everything works |
| 3 | [Exploring the Sample Data](03-sample-data) | How the synthetic dataset is structured and what each variable represents |
| 4 | [Running the Analysis Pipeline](04-run-pipeline) | How to execute all 10 analysis scripts and interpret the pipeline log |
| 5 | [Interpreting Your Results](05-results) | How to read the output tables, figures, and statistical tests |
| 6 | [Adapting to Your Study](06-adapt) | How to replace sample data with your own, customize instruments, and adjust parameters |
